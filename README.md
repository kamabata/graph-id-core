

# Graph ID

## Installation 

### From pypi

```
pip install graph-id-core
pip install graph-id-db # optional
```

Test on Ubuntsu and Google Colaboratory

### From GitHub

For MacOS users

```
git clone git+https://github.com/kmu/graph-id-core.git
git submodule init
git submodule update
pip install -e .
```
